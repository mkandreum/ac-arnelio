## ¿Qué es AC Arnelio?

AC Arnelio es una plataforma web desarrollada por Xyon Platforms para la gestión de comunidades de vecinos y administración de fincas. Centraliza la comunicación, incidencias, documentación y gestión económica de la comunidad en una sola herramienta.

## ✨ Funcionalidades

- **Gestión de incidencias** y comunicados
- **Portal de vecinos** con acceso a documentación
- **Gestión económica** y cuotas
- **Tablero de anuncios** de la comunidad
- **Panel de administración** para el administrador de fincas
- **Despliegue self-hosted** vía Docker y Coolify

## 🛠️ Stack técnico

| Capa | Tecnología |
|------|----------|
| Frontend | React + TypeScript + Vite |
| Backend | Node.js + TypeScript |
| Despliegue | Docker + Coolify |

## 🚀 Instalación

```bash
git clone https://github.com/mkandreum/ac-arnelio.git
cd ac-arnelio
cp .env.example .env
docker-compose up -d
```

### Desarrollo local

```bash
npm install
npm run dev
```
